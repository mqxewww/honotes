## How To Use

To run this application, you'll
need [Bun](https://bun.com/docs/installation). When everything is installed, you can run from your command line:

```bash
# Clone this repository
$ git clone https://github.com/mqxewww/honotes.git

# Install dependencies
$ bun install

# Setup environment variables (development or production)
$ cp .env.template .env.{development,production}

# Run the app
$ bun run {dev,start}
```
